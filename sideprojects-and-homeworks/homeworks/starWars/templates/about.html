About the project

This Star Wars Info/Fan website is a result of a merger of two separate and different tasks from SEDC Academy of Web Development, one from the subject Basic JavaScript Development and the other from the subject Advanced JavaScript Development. The original idea was to merge the both codes into one SPA app, but since it needed more refactoring and optimization, I opted for the multiple pages approach, with only adjusting the style and UX to be more compatible since the idea is that they are both different pages from one website.

The whole credits for the CSS code of this crawler go to Geoff Graham. css-tricks.com/snippets/css/star-wars-crawl-text
